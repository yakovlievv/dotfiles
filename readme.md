# Dotfiles

Personal configuration for macOS and Arch Linux.

## Installation

> [!WARNING] Before cloning you must set up ssh to connect to github.

Clone the repo:

```bash
git clone git@github.com ~/dotfiles
cd ~/dotfiles
```

Run the setup script for your system:

Arch Linux

```bash
./install-arch.sh
```

macOS

```bash
./install_mac.sh
```

The scripts will install all dependencies and link the configs automatically.
